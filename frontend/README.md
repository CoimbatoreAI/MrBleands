# Mr Blends

Authentic Kerala Pickles – Homemade Taste in Every Jar.

## Project info

Mr Blends is a premium Kerala pickle brand offering traditional handcrafted pickles made with fresh ingredients and traditional recipes.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd MrBlends

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
